# YouTube to MP3 Downloader

**Safe, high-quality, spam-free YouTube to MP3 converter script**.  
This project demonstrates **Bash scripting, automation, and working with external tools** like `yt-dlp` and `ffmpeg`.

---

## Features

- Works with **any YouTube video**
- Converts automatically to **high-quality MP3**
- Handles **clips, shorts, and standard videos**
- Saves files locally for **offline listening**
- **Spam-free** alternative to sketchy online converters

---

## Technical Skills Demonstrated

- Bash scripting and command-line interface design
- File management and automation
- Integration with third-party tools (`yt-dlp`, `ffmpeg`)
- Cross-platform considerations (Linux, WSL, macOS)

---
